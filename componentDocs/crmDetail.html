    # crmDetail

    详细信息组件

    # 示例

    详细信息
    广州佛山同圣科技软件有限公司
    线上注册
    18703850038
    68873988
    jiaxun000@gmail.com
    https://www.28kcrm.com/base

    # 代码

    <template>
  <div id="app">
    <crm-detail :detail-items="items" :detail-title="title"> </crm-detail>
  </div>
</template>

<script>
  export default {
    data() {
      return {
        title: "基本信息",

        items: [
          {
            label: "客户名称",
            value: "广州佛山同圣科技软件有限公司",
            type: "text",
          },
          { label: "客户来源", value: "线上注册", type: "text" },
          { label: "手机", value: "18703850038 ", type: "text" },
          { label: "电话", value: "68873988", type: "text" },
          { label: "邮箱", value: "jiaxun000@gmail.com", type: "text" },
          {
            label: "网址",
            value: "https://www.28kcrm.com/base ",
            type: "text",
          },
        ],
      };
    },
  };
</script>

    # Attributes

    参数 说明 类型 可选值 默认值
    detail-items 显示的数据 Array - -
    detail-title 标题 String - 基本信息
